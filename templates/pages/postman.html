LeanCloud × Postman

Postman 版本的 REST API 调用工具箱

LeanCloud × Postman 是一个帮助开发者快速了解 LeanCloud REST API、调用 API 的工具。通过 Postman 的 Google Chrome 扩展实现。
特性：

  • 可以灵活设置 URL，URL Get 参数， HTTP header 参数， JSON 格式数据。
  • 可以把发送的请求转换为 cURL 等语言的实现。
  • 可以通过 Google Chrome 开发者工具查看实际发出的请求和响应

关于 Postman

Postman 是一个模拟 HTTP 请求的工具，方便开发者快速构建、调试 API。

如何使用

  1. 安装 Postman，Postman 是一个 Chrome 扩展。
  2. 引入 LeanCloud Collection 文件。定义了 LeanCloud REST API 调用格式的文件。
  3. 引入 LeanCloud 全局环境变量文件，定义了 host、API 版本的全局变量。
  4. 引入 LeanCloud 应用级别环境变量文件，定义了应用的 AppID，AppKey，Master Key。可以替换成开发者自己的应用的相关值
